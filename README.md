# Golang CSV MongoDB Wilayah Administrasi Indonesia
> A starter project with Golang

## Installation

__Install Glide__

`https://github.com/Masterminds/glide`

then run:

```
glide install
```

## Running The App

Just type :

```
go run main.go
```
